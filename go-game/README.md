# 围棋游戏 (9x9)

这是一个使用HTML、CSS和JavaScript实现的9x9围棋游戏，支持人机对战，具有不同难度级别。

## 游戏特性

- 9x9标准围棋棋盘
- 三种难度级别（简单、中等、困难）
- 完整的围棋规则实现
- 简约美观的界面设计
- 支持虚手和重新开始功能

## 如何运行

1. 确保您的系统已安装Python 3
2. 在项目目录下运行以下命令启动服务器：
   ```bash
   python3 -m http.server 8000
   ```
3. 打开浏览器并访问 `http://localhost:8000`

## 游戏规则

- 玩家执黑子，电脑执白子
- 点击棋盘交叉点放置棋子
- 轮到您时，可以选择"虚手"跳过回合
- 游戏结束时，系统将自动计算领地并显示胜负结果

## 文件结构

- `index.html` - 游戏的主界面
- `style.css` - 样式定义
- `script.js` - 游戏逻辑和AI实现
